Delicious est un site vitrine de recette de cuisine réalisé avec NextJS. Il utilise l'api Gastronogeek pour récupérer et lister les différentes recettes. Ce site à été réalisé lors d'un exercice pendant ma formation en master pour nous introduire à ce framework. Également, nous avons été introduis aux transitions de page et incité à utiliser des animations sur les éléments de la page. Ce site à également été l'opportunité pour moi d'utiliser une charte graphique pensée lors d'un autre projet mais que je n'avais jamais encore utilisé.
